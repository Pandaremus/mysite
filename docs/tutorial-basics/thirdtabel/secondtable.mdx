# 总体结构分析

:::note  主要架构
 1.HKEY_CLASSES_ROOT<br/>
 说明：该根键包括启动应用程序所需的全部信息，包括扩展名，应用程序与文档之间的关系，驱动程序名，DDE和OLE信息，类ID编号和应用程序与文档的图表等。<br/>

 2.HKEY_CURRENT_USER<br/>
 说明：该根键包括当前登录用户的配置信息，包括环境变量，个人程序以及桌面设置等。<br/>

 3.HKEY_LOCAL_MACHINE<br/>
 说明：该根键包括本地计算机的系统信息，包括硬件和操作系统信息，安全数据和计算机专用的各类软件设置信息。<br/>

 4.HKEY_USERS<br/>
 说明：该根键包括计算机的所有用户使用的配置数据，这些数据只有在用户登录系统时才能访问。这些信息告诉系统当前用户使用的图标，激活的程序组，开始菜单的内容、颜色以及字体。<br/>
 
 5.HKEY_CURRENT_CONFIG<br/>
 说明：该根键包括当前硬件的配置信息，其中的信息是从HKEY_LOCAL_MACHINE中映射出来的。<br/>

:::
