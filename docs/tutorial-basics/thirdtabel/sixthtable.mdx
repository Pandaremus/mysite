# 注册表破坏的现象及原因

:::note 注册表破坏后的常见现象
 1、无法启动系统。<br/>
 2、无法运行或正常运行合法的应用程序。<br/>
 3、找不到启动系统或运行应用程序所需的文件。<br/>
 4、没有访问应用程序的权限。<br/>
 5、不能正确安装或装入驱动程序。<br/>
 6、不能进行网络连接。<br/>
 7、注册表条目有错误。<br/>
:::


:::note 注册表被破坏的原因
 1、应用程序错误：
此种情形出现的比较多，因为应用程序或多或少的会出现错误，都可能导致不同的后果；另外，在安装过多软件之后，有可能出现彼此之间的冲突。<br/>
 2、驱动程序不兼容：
此种情形出现的也较多，如果本身有驱动程序盘的话，建议用专业版。<br/>
 3、硬件问题：
   此处的问题主要是出现在硬件质量上，比如硬盘或内存质量不过关造成读写错误，或者是超频、CMOS、病毒等。<br/>
 4、误操作：
   误操作可能会导致注册表出现错误，有的时候甚至会很严重。<br/>
