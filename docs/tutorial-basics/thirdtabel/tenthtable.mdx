# 个性化设置
:::danger 以下需要重启计算机
1、**去掉桌面快捷方式上的小箭头**<br/>
位置：HKEY_CLASSES_ROOT\LNKFILE项 ：Is Short cut<br/>
操作：删除此项。<br/>

2、**隐藏桌面上的所有图标**<br/>
位置：HKEY_CURRENT_USER\Software\Windows\CurrentVersion\Policies\Explorer<br/>
新建：No Desktop（双字节）。<br/>
操作：值设为1。<br/>

3、**修改桌面上“回收站”的名字及图标**<br/>
位置：HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}（这个大家也可以直接查找“回收站”）。<br/>
修改：（默认）一项进行名字修改。然后展开此项，选择DefaultIcon，这时大家看到有3个子项，如果想改变图标时，只需将路径和文件名改了即可。<br/>

4 、**去掉桌面上的网上邻居**<br/>
位置：HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer<br/>
新建：NoNetHood（双字节）值： 1<br/>

5、**在桌面上显示系统版本号**<br/>
位置：HKEY_CURRENT_USER\Control Panel\Desktop<br/>
项：PaintDesktopVersion<br/>
修改：将值改为1<br/>

6、**改变窗口按钮字体的颜色**<br/>
位置：HKEY_CURRENT_USER\Control Pannel\Colors<br/>
键值：Button Text<br/>
修改：可以像PS一样，比如改为红(255,0,0)<br/>

7、**屏蔽桌面上的IE图标**<br/>
位置：HKEY_CURRENT_USER\Software\Windows\CurrentVersion\Policies\Explorer<br/>
新建值项：No InternetIcon<br/>
修改：1<br/>

8、**禁止更改桌面墙纸**<br/>
位置：HKEY_CURRENT_USER\Software\Windows\CurrentVersion\Policies\Explorer<br/>
新建项值：No Changing Wall Paper(双字节）<br/>
修改：1<br/>

9、**设置登录背景**<br/>
位置：HKEY_USERS\DEFAULT\ControlPanel\Desktop<br/>
键值：WallPaper<br/>
修改：把背景图片一写就OK了（比如C:\aa.bmp）<br/>
:::
