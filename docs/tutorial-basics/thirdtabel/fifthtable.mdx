



# 注册表的基本操作


:::note 基本操作
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1、创建项和项值。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2、更值项的数据。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3、删除项、子项或值项。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;4、查找项、值项或数据。<br/>
 注册表中常用的数据类型有5种：<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**二进制值（reg_binary）**:多数硬件信息以二进制数据存储，而以十六进制格式显示在注册表编辑器中。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**字符串值（reg_sz）**:包括字符串的注册表键，使用字符串数据类型。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**双字节值（reg_dword）**:是32位信息常显示成4个字节。它在出错控制功能上用处极大，其数据一般以十六进制格式显示在注册表编辑器中。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**多字符串值(reg_multi_sz)**:允许将一系列项目作为单独的一个值使用。对于多种网络协议、多个项目、设备列表以及其他类似的列表项目来说，可以使用多字符串值。<br/>
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**可扩充字符串值（reg_expand_sz)**：代表一个可扩展的字符串。<br/>
