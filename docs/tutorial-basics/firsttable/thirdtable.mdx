# 插件安装
### Falsh安装


:::note ~
 flash官方已经不提供falsh支持，所以我们从中国重庆重橙网络科技有限公司为广大国内用户提供“特供版”的FlashPlayer处下载。
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/10.png" />
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/11.png" />
:::

:::danger 注意
 业务管理系统使用的浏览器为低版本火狐浏览器，因此在安装flash时选择第三个
<BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/12.png" />
:::

:::note ~
 安装软件时由于flash官方已经关闭flash下载，所以目前安装的所有的flash都会被windows认定为盗版软件，包括中国“特供版”falsh，所以在安装完flash之后，windows中心会进行预警，并试图阻止flash正常运行并删除主要程序目录。
<BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/13.png" />
:::


:::note ~
 我们需要点击操作，将flash手动添加为信任。这样下来，falsh即可正常运行。
<BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/14.png" />
:::





:::danger 注意
业务管理系统上传报告两个必备条件：1、使用低版本火狐浏览器，安装包位于企业微信微盘。2、安装flash插件。**[点击此处下载flash安装包](ispackage/flashplayer_install_cn_web.exe)**
:::


## web office安装、修复


:::note ~
iWebOffice2015智能文档中间件支持多种不同电子文档格式，可支持doc、docx、xls、xlsx、wps、et等格式文档；支持微软Office，金山Wps，永中Office等软件在线编辑；支持另存为PDF、HTML、TXT等格式文档；支持文档部分区域保护，禁止使用指定快捷等功能。

iWebOffice2015支持主流多浏览器应用，包括微软IE、Edge、Google Chrome、Firefox、360极速/安全浏览器等，都可适配运行。

iWebOffice2015新一代基于64位架构，可提供强劲的运算，同时支持32位和64位运算。支持32/64位IE浏览器、32/64位OFFICE办公软件、32位/64位Windows操作系统。 
:::
