# 软件安装


:::note **定义** 
 软件安装在这里指的是在Windows系统下通过软件的安装程序进行安装，不包括软件安装（例如在steam上安装游戏）以及免安装软件，所安装的软件一律是正版软件，不包括破解版。
:::

:::note **软件来源**
 安装软件的关键在于运行软件的安装程序，这是一个可执行文件(后缀名为.exe的文件)，这个程序可以自动将软件安装到计算机上。我们可以通过许多渠道获得软件的安装程序以及配套的文件，但是需要仔细甄别，从不正规渠道获得的软件安装包可能会含有意想不到的“附加内容”。一般来说，首先选择的应该是软件厂商的官方网站，以安装迅雷下载器为例。首先在搜索引擎搜索软件名称，得到如下结果。

:::danger **注意**
  找到标注有“官方”字样的链接，这表明这是由搜索引擎运营商（百度）认证过的官方网站。如果没有这样的标注，也可以通过观察网站域名来甄别，一般来说域名为[www.xxxxxx(网站的英文名或者缩写).(edu).com]或者[www.xxxxxx.(edu).cn]的网站有更大的可能性是官方网站(与教育相关的网站可能带有edu字样)。其次，官方网站的特点往往是简洁风格，如果网页有大量广告、浮窗，则可能不是官方网站。如果官方网站无法提供下载，建议在正规软件下载平台进行下载，例如softonic平台。

:::note **软件的安装目录**
 选择安装目录的选项。可以看到，程序安装的目录一般都是在C盘的Program Files文件夹下面。然而，这种目录一般是不推荐，C盘属于系统盘，在计算机正常运转过程中需要在C盘划定一块虚拟内存，虚拟内存的大小影响计算机的运行速度，如果C盘空间被安装的程序过多占用，可能导致计算机运行速度下降；其次，在系统盘安装软件也增加了流氓软件修改系统文件的可能性，增大了计算机出现问题的风险。

 **一些免费软件，往往会为了推广其他厂商或者本厂商的产品，在软件的安装程序中嵌入附加安装选项。所有在安装软件的时候务必仔细观察，认真的阅读软件安装过程中的每一个界面将附加选项取消勾选，开始安装。**
