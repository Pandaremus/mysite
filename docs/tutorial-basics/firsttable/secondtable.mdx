# 软件卸载

:::note **软件安装目录卸载**
 软件的安装目录下面有一个叫做卸载程序的可执行文件，可以自动完成卸载软件的步骤。这个软件一般命名为uninstall.exe, uninstaller.exe。运行这个软件就可以完成卸载。
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/4.png" />
:::


:::note **控制面板卸载**
 在电脑屏幕下方[搜索]栏目输入控制面板，找到卸载程序
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/5.png" />
:::

:::note ~
 找到需要卸载的软件右击卸载
  <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/6.png" />
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/7.png" />
:::


:::note **手动卸载**
 找到软件的安装目录，选中以后按住“shift+delete"键（先按住shift再按下delete）进行彻底删除（不经过回收站的那种删除）。
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/8.png" />
::: 



:::note ~
 然后打开注册表编辑器，找到注册表文件夹。右键删除整个文件夹。
 <BasicImageViewer src="https://docusaurus-1315095412.cos.ap-beijing.myqcloud.com/image/DocsFirstCategory/9.png" />
:::
